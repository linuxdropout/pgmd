# pgmd Frontend

Static SPA scaffold with strict TypeScript, strict ESLint, SCSS, Jotai state, Zod validation,
Vitest unit tests, Playwright e2e tests, Prettier, Stylelint, and Knip.

## Scripts

- `npm run dev`: start Vite dev server
- `npm run build`: type-check and build production assets
- `npm run preview`: preview production build locally
- `npm run typecheck`: run TypeScript project checks
- `npm run lint`: run strict ESLint
- `npm run lint:fix`: apply ESLint autofixes
- `npm run format`: format with Prettier
- `npm run format:check`: verify Prettier formatting
- `npm run stylelint`: lint SCSS files
- `npm run stylelint:fix`: apply Stylelint autofixes
- `npm run test`: run Vitest
- `npm run test:coverage`: run Vitest with coverage
- `npm run test:e2e`: run Playwright e2e tests
- `npm run knip`: detect unused files, dependencies, and exports

## Environment

- `VITE_APP_TITLE` (optional): heading text displayed in the app. Defaults to `pgmd`.
